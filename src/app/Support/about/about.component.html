Architecture

The architecture of PRS is full-stack, three tier with web front-end, Microsoft .NET back-end, and SQL server for the data

Business Flow

The business flow in PRS is based on the purchase request.

There are users to limit the system operation to only authorized users.

Technology

The technologies used on PRS are the following:

  • HTML/CSS
  • JavaScript
  • Bootstrap
  • jQuery
  • Typescript
  • Angular
  • C#
  • EntityFramework
  • MVC
  • WebAPI
  • SQL Server
